## About the Project
This is my personal portfolio showcasing my expertise as a **Meteorologist and Climate Researcher**. It features high-performance animations, a real-time weather integration, and a gallery of atmospheric fieldwork.

## Features:
- **Real-time Weather**: Live data fetching using Open-Meteo API.
- **Interactive Gallery**: Smooth, auto-scrolling field trip highlights.
- **Modern UI**: Built with Next.js, Framer Motion, and Tailwind CSS.
- **Responsive**: Fully optimized for mobile and desktop viewing.
